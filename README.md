## 🚀 Getting Started

### 📦 Dependency

- node v18+
- npm v9.5+
- gulp v4+

### 👉 Package Installation

```bash
npm install
```

### 👉 Development Command

```bash
npm run dev
```

### 👉 Build Command

```bash
npm run build
```
